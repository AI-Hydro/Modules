# Capstone: A Basin's Water Story

**Level:** Intro · **Time:** ~60 min · **License:** CC-BY-4.0 · **Course:** [Introduction to Hydrology](../intro-to-hydrology/)

Synthesis capstone for the Intro to Hydrology course. Choose one of four curated basins (or any USGS gauge), fetch a 10-year streamflow record, compute hydrological signatures (BFI, FDC), run a TWI terrain analysis, then narrate the basin's water story through a 6-stage interactive timeline. Generates a structured text report you can export.

## What you'll learn

See the module itself — open it in AI-Hydro's HTML Preview panel.

## Requirements

Python packages: `numpy`, `matplotlib`, `dataretrieval` _(optional — falls back to synthetic data if absent)_

## Citation

Addor, N. et al. (2017). The CAMELS data set: catchment attributes and meteorology for large-sample studies. Hydrol. Earth Syst. Sci., 21, 5293–5313.
