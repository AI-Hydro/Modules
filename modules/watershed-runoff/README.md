# The Watershed: Where Rain Becomes Runoff

**Level:** Intro · **Time:** ~40 min · **License:** CC-BY-4.0 · **Course:** [Introduction to Hydrology](../intro-to-hydrology/)

Define a catchment, trace D8 flow paths across a 3-D draggable terrain, and drop a virtual rain cloud over the basin to watch the hydrograph respond. Covers watershed boundaries, flow accumulation, and the link between contributing area and streamflow.

## What you'll learn
