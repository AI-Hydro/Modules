# Catching the Flow: Real Streamflow Data

**Level:** Intro · **Time:** ~45 min · **License:** CC-BY-4.0 · **Course:** [Introduction to Hydrology](../intro-to-hydrology/)

Pull live USGS records for four curated CONUS gauges spanning flood-prone Maine to arid Arizona — or enter any gauge ID. Plot the annual regime, spot a flood event, and read a flow-duration curve. Falls back to seeded synthetic data when the network is unavailable.

## What you'll learn

See the module itself — open it in AI-Hydro's HTML Preview panel.

## Requirements

Python packages: `numpy`, `matplotlib`, `dataretrieval` _(optional — falls back to synthetic data if absent)_

## Citation

USGS National Water Information System (NWIS). https://waterdata.usgs.gov/
