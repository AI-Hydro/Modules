# Reading a Hydrograph

**Level:** Intro · **Time:** ~30 min · **License:** CC-BY-4.0 · **Course:** [Introduction to Hydrology](../intro-to-hydrology/)

Anatomy of a storm hydrograph — rising limb, peak, recession — with an interactive BFI explorer, a storm-pair comparison revealing non-linear runoff response, and a live Python cell that generates CAMELS-style annual hydrographs for three contrasting gauges.

## What you'll learn

See the module itself — open it in AI-Hydro's HTML Preview panel.

## Requirements

Python packages: `numpy`, `matplotlib`

## Citation

Lyne, V. & Hollick, M. (1979). Stochastic time-variable rainfall-runoff modelling. Inst. Eng. Australia Nat. Conf. Publ. 79/10, 89–93.
