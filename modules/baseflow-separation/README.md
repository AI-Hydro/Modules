# Splitting the Flow: Baseflow Separation

**Level:** Intro · **Time:** ~40 min · **License:** CC-BY-4.0 · **Course:** [Introduction to Hydrology](../intro-to-hydrology/)

Run the Lyne–Hollick digital filter interactively — drag the α slider to rewrite the live Python cell in real time, compare filtered vs raw hydrographs side by side, and understand why the Baseflow Index matters for drought assessment and groundwater recharge estimation.
